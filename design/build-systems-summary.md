# 빌드 시스템 요약 — 무기 & 패시브

> **버전**: 2026-04-28
> **상태**: MVP 구현 완료 (UI 패널 제외)
> **공유 대상**: 팀원
> **출처 GDD**: `weapon-system.md`, `spec-system.md`, `levelup-selection.md`, `weapon-selection-ui.md`

매 런마다 **무기 3슬롯 + 패시브 12종** 조합으로 빌드를 만든다. 레벨업 시 무기·패시브 혼합 카드 3장 중 1장을 선택. 각 카드는 등급(커먼/에픽/유니크/레전드)을 가지며, 등급에 따라 1회 선택의 수치 폭이 차등.

---

## 1. 무기 시스템

### 1-1. 무기 3종

| 무기 | 패턴 | 특징 |
|------|------|------|
| **검 (Sword)** | 주변 부채꼴 베기 | 근거리 광역, 마일스톤마다 각도 확장 (180° → 360°) |
| **총 (Gun)** | 고속 직선 투사체 | 원거리 단일 표적, 마일스톤마다 관통 횟수 증가 (0 → 무한) |
| **마법 (Magic)** | 느린 투사체 + 폭발 | 원거리 광역, 마일스톤마다 폭발 반경 확장 (2m → 7m) |

### 1-2. 무기 슬롯 규칙

- **최대 슬롯**: 3개. 가득 차면 새 무기 추가 카드 등장 안 함 (강화만 가능)
- **자동 발동**: 각 슬롯이 독립 타이머로 동작 (인터벌은 무기 데이터 + 패시브로 결정)
- **레벨**: 무기당 **최대 Lv 15**. 동일 무기 카드 재선택 시 +1
- **모든 무기 + 슬롯 가득 + 모두 Lv 15** 도달 시 무기 카드는 등장 안 함 (패시브만)

### 1-3. 무기 강화 효과 (레벨업)

| 레벨업 시 변화 | 적용 |
|---------------|------|
| **데미지** | 매 레벨 등급별 가중치만큼 누적 |
| **공격 인터벌** | 매 레벨 등급별 가중치만큼 단축 |
| **범위 (각도/사거리/폭발 반경)** | **마일스톤(Lv 1, 5, 10, 15)에서만** 점프 |
| **관통 (총 전용)** | 마일스톤(0 → 1 → 2 → 무한) |

### 1-4. 마일스톤 테이블

| 무기 | Lv1 | Lv5 | Lv10 | Lv15 |
|------|-----|-----|------|------|
| 검 — 부채꼴 각도 | 180° | 240° | 300° | 360° |
| 검 — 사거리 배율 | ×1.0 | ×1.4 | ×1.8 | ×2.5 |
| 총 — 사거리 배율 | ×1.0 | ×1.4 | ×1.8 | ×2.5 |
| 총 — 관통 횟수 | 0 | 1 | 2 | 무한 |
| 마법 — 사거리 배율 | ×1.0 | ×1.4 | ×1.8 | ×2.5 |
| 마법 — 폭발 반경 | 2.0m | 3.0m | 4.5m | 7.0m |

## 2. 패시브 시스템 (12종)

레벨업 카드에서 무기와 함께 등장. 각 패시브 **최대 Lv 15** (선택 횟수 기준).

### 2-1. 생존 계열 (3종)

| 패시브 | 효과 | Lv1 (커먼 1회) | Lv15 누적 (커먼만) |
|--------|------|----------------|---------------------|
| **최대 체력** | MaxHp +15/Lv | +15 HP | +225 HP |
| **HP 회복** | 초당 자동 회복 | +0.5/s | +7.5/s |
| **회피** | 데미지 무효화 확률 (DR cap 60%) | 8.0% | 70.6% |

### 2-2. 공격 계열 (4종)

| 패시브 | 효과 | Lv1 (커먼 1회) | Lv15 누적 (커먼만) |
|--------|------|----------------|---------------------|
| **공격 속도** | 인터벌 ×0.95^Lv (floor 0.30), 발사체/검 모션 속도 동시 ↑ | 인터벌 ×0.95 | ×0.46 (2.17배 빨라짐) |
| **치명타 확률** | 크리 발생률 (DR cap 75%) | 8.0% | 88.0% (cap 75%) |
| **치명타 피해** | 크리 데미지 배율 +0.15/Lv | +15% | +225% |
| **생명 흡수** | 데미지의 N% HP 회복 (cap 25%) | +2% | +25% (cap) |

### 2-3. 투사체/공격 (1종)

| 패시브 | 효과 | Lv1 | Lv15 |
|--------|------|-----|------|
| **발사체 수** | 3Lv마다 +1 (총·마법=발사 수, 검=연속 휘두르기) | 0 | +5 |

> **발사체 속도**는 패시브가 아닌 **공격 속도의 파생 효과**로 통합됨. 공격 속도가 빨라지면 발사체 이동 속도/검 휘두르기 속도도 같이 빨라진다.

### 2-4. 이동 계열 (2종)

| 패시브 | 효과 | Lv1 | Lv15 누적 (커먼만) |
|--------|------|-----|---------------------|
| **이동 속도** | MoveSpeed +1.0/Lv (베이스 6.0) | +1.0 | +15.0 |
| **추가 점프** | 공중 추가 점프 +1/Lv | +1 (더블점프) | +15 |

### 2-5. 메타 계열 (2종)

| 패시브 | 효과 | Lv1 | Lv15 |
|--------|------|-----|------|
| **행운** | 레벨업 카드 고등급 가중치 (DR cap 80%) | 10% | 80% (cap) |
| **난이도** | 적 스폰 +15%/Lv, XP/Gold 보상 +20%/Lv | spawn ×1.15 / reward ×1.20 | spawn ×3.25 / reward ×4.00 |

## 3. 등급 시스템

레벨업 카드 등장 시마다 등급이 랜덤 결정. **레벨은 항상 +1**, 등급은 **수치 폭만 차등**.

### 3-1. 등장 확률 (행운 0 기준)

| 등급 | 색상 | 기본 확률 |
|------|------|-----------|
| 커먼 | 회색 | 60% |
| 에픽 | 보라 | 25% |
| 유니크 | 노랑 | 12% |
| 레전드 | 빨강 | 3% |

행운 패시브 누적 시 커먼 비중이 줄고 상위 등급 비중이 증가 (DR cap 80%).

### 3-2. 등급별 효과 (1회 선택의 수치 폭)

#### 무기

| 등급 | 데미지 +bonus | 공격 인터벌 −reduction |
|------|--------------|------------------------|
| 커먼 | +12% | −6% |
| 에픽 | +20% | −10% |
| 유니크 | +32% | −15% |
| 레전드 | +50% | −22% |

> 매 레벨업마다 누적. 예: 레전드 5번 = 데미지 +250%

#### 패시브

레벨은 항상 +1이지만, 수치 환산용 **effective level**에는 등급별 가중치가 누적된다.

| 등급 | effective level 가중치 |
|------|------------------------|
| 커먼 | ×1.0 |
| 에픽 | ×1.5 |
| 유니크 | ×2.5 |
| 레전드 | ×4.0 |

> 예: HpRegen 레전드 1회 선택 = effective Lv 4.0 → +2.0 HP/s
> (커먼 4회 선택과 같은 수치, 단 1회 선택만으로 도달)

### 3-3. 동작 사례

```
[상황] HpRegen 패시브 4회 선택, 등급은 모두 다름
- 커먼 1회 → effLv 1.0 → +0.5 HP/s
- 에픽 1회 → effLv 2.5 → +1.25 HP/s (누적)
- 유니크 1회 → effLv 5.0 → +2.5 HP/s (누적)
- 레전드 1회 → effLv 9.0 → +4.5 HP/s (누적)

picks = 4 (Lv 15 cap 판정용)
effLv = 9.0 (수치 산출용)
```

## 4. 레벨업 선택 흐름

### 4-1. 트리거
- 적 처치 → XP 획득 → 레벨업 발생 (`XPSystem.OnLevelUp`)
- 또는 상자 구매 (추후)

### 4-2. 동작
1. `Time.timeScale = 0` — 전투 완전 정지
2. 카드 3장 빌드 (무기 + 패시브 풀에서 무작위)
3. UI 표시 (현재 미구현 — `WeaponSelectionUI.cs` 코드만 존재)
4. 플레이어 1/2/3 키 또는 클릭으로 선택
5. 무기면 `WeaponSystem.AddWeapon/UpgradeWeapon` 호출
6. 패시브면 `PlayerStats.IncrementPassive(type, grade)` 호출
7. `Time.timeScale = 1` — 전투 재개
8. 연속 레벨업은 큐잉

### 4-3. 카드 풀 규칙
| 조건 | 카드 등장 |
|------|-----------|
| 미보유 무기 + 슬롯 여유 | 신규 무기 |
| 보유 무기 + Lv 15 미만 | 무기 강화 |
| 미보유 또는 Lv 15 미만 패시브 | 신규 또는 패시브 강화 |
| 모든 항목 최대 | 카드 0장 → 즉시 재개 |

## 6. 키 매핑 (현재 디버그 모드)

| 키 | 동작 |
|----|------|
| **W/A/S/D** | 이동 |
| **Space** | 점프 (ExtraJump 패시브로 다중 점프 가능) |
| **Shift** | 대시 (무적 0.2초) |
| **1/2/3** | (WeaponDebugger) 검/총/마법 추가·강화 |
| **4~8** | (WeaponDebugger) 등급 모드 변경 |
| **0** | (WeaponDebugger) 행운 패시브 +1 |
| **M** | MoveSpeed +1 |
| **J** | ExtraJump +1 |
| **H** | MaxHp +1 |
| **V** | Dodge +1 |
| **Y** | HpRegen +1 |
| **K** | AttackSpeed +1 (발사체 속도/검 모션 속도 포함) |
| **Q** | CritChance +1 |
| **E** | CritDamage +1 |
| **T** | Lifesteal +1 |
| **U** | ProjectileCount +1 |
| **O** | Luck +1 |
| **P** | Difficulty +1 |
| **R** | 게임 재시작 (Scene 리로드) |
| **Esc** | 일시정지/재개 |

> 실제 출시 빌드에서는 디버거 컴포넌트 모두 제거 예정. 1~9, 0, Q/E/T/Y/U/I/O/P/H/J/K/M/V는 게임플레이용으로 비워둠.
