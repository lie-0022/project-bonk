## Overview

무기 선택 UI는 레벨업 시 표시되는 무기 3택 1 팝업 화면이다. 화면 중앙에 3개의 무기 카드를 가로로 나열하고, 각 카드는 무기 이름·설명·등급·강화 여부를 표시한다. 플레이어가 클릭하면 선택 결과를 레벨업 무기 선택 시스템으로 전달한다. MVP에서는 단순 버튼 3개로 구현한다.

## Player Fantasy

깔끔하고 즉각적인 선택 화면. 어떤 무기인지, 새 무기인지 강화인지 한눈에 파악할 수 있어야 하고, 클릭하는 순간 바로 전투로 복귀해야 한다.

## Detailed Design

### Core Rules

1. `LevelupWeaponSelection.OnSelectionRequired` 이벤트 수신 시 패널 활성화
2. 선택지 수만큼 카드 버튼 표시 (최대 3개)
3. 각 카드 표시 내용:
   - 무기 이름
   - 등급 (색상으로 구분: 커먼=회색, 에픽=보라, 유니크=노랑, 레전드=빨강)
   - 한 줄 설명 (공격 패턴 요약)
   - **신규** 또는 **강화 Lv X → Lv X+1** 표시
4. 카드 클릭 시 `LevelupWeaponSelection.OnWeaponChosen(weaponType)` 콜백 호출
5. 콜백 후 패널 비활성화
6. 키보드 단축키: 1번 카드=1키, 2번 카드=2키, 3번 카드=3키

### Interactions with Other Systems

- **← 레벨업 무기 선택**: 선택지 데이터 수신, 선택 결과 반환

## Edge Cases

- **선택지 2개**: 카드 2개만 표시, 3번째 슬롯 비활성화
- **선택지 1개**: 카드 1개만 표시
- **빠른 클릭**: 클릭 후 즉시 버튼 비활성화해 중복 선택 방지

## Visual/Audio Requirements

- 카드: 등급 색상 테두리, 무기 이름(굵게), 설명(작게), 신규/강화 뱃지

## Acceptance Criteria

- [ ] 레벨업 시 화면 중앙에 무기 카드가 표시됨
- [ ] 카드 클릭 또는 1/2/3 키 입력으로 선택 가능
- [ ] 선택 후 UI가 즉시 사라지고 전투가 재개됨
- [ ] 각 카드에 무기 이름, 설명, 등급 색상이 표시됨
- [ ] 강화 선택지는 현재 레벨과 강화 후 레벨이 표시됨
- [ ] 신규 무기는 "신규" 뱃지가 표시됨
