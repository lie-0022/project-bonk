---
name: 파일 저장 사전 승인
description: 사용자가 파일 저장 시 매번 확인 질문을 생략해도 된다고 명시적으로 허용함
type: feedback
---

파일을 저장하기 전에 "저장해도 될까요?" 같은 확인 질문을 하지 않아도 됨. 각 GDD 섹션마다 "괜찮으신가요?" 도 묻지 않아도 됨. 사용자가 무조건 허용한다고 명시적으로 말함.

**Why:** 매번 승인을 묻는 것이 워크플로우를 느리게 만듦. 사용자가 빠른 진행을 원함.

**How to apply:** GDD 섹션 설계 시 초안을 보여주고 바로 저장. 파일 경로는 명시적으로 알려줄 것. 큰 방향 결정(예: 탑다운 vs 아이소메트릭 선택)은 여전히 질문해야 함 — 승인 확인이 아닌 설계 결정 질문은 계속 필요.
